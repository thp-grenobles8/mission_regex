/REX-REGEX/

ou comment devenir le roi des expressions régulières

introduction

Ce petit tutoriel est là pour t'apprendre à utiliser les "expressions régulières" aussi appelées regex ou regexp.

Regex n'est pas un langage en soi (@Lucien avait raison au final) mais une grammaire, un ensemble de codes que l'on retrouve dans la pluspart des langages de programmation (Ruby, Python et Javascript à minima).

Dans ces trois langage, la convention veut qu'on écrive les regex entre slashs comme ça :

/bonjour/

comment les utiliser

Les regex servent à matcher des bouts de texte très précis, de manière rapide et efficace.

quelques exemples :

    # en Ruby ------------------
    "hello my dear".match(/my/)
      #retourne true car "my" est dans la string
    ""
